# Command Station Prototype Assets

These original generated bitmap assets are used by `public/command-station-demo.html`.

They are prototype/runtime assets for the standalone proof page, not final production UI for the game client.

## Files

- `console-shell-v2.png` - fixed command-room/mech-bay style shell with blank monitor apertures.
- `sensor-map-v2.png` - painterly tactical space-map plate used inside the central sensor display.
- `commander-portrait-v2.png` - fictional commander uplink portrait used inside the right monitor.

## Direction

The page should feel like a permanent authored machine surface with animated screens embedded into it. Do not rebuild this direction from visible web cards or generic dashboard panels.

## Generation Notes

Generated with the built-in image generation tool as original non-infringing prototype assets. Prompts asked for a 1990s pre-rendered sci-fi bitmap feeling with modern polish and explicitly avoided copied franchise artwork, readable text, logos, low-poly battle scenes, and existing characters.

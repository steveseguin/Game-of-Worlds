Quick Help

  • Turns advance every 3 minutes; use the Next Turn button when ready.
  • Resources: build extractors/refineries on owned sectors; research unlocks tech and better ships.
  • Colonize: build a colony ship (Fleet tab), move it to an unowned sector, then colonize.
  • Combat: fleets in the same sector auto-resolve each turn; build scouts early for vision.
  • Need a refresher? Reopen this help with the ? button.
